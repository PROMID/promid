Track Description

PROMID: Prompt RecOvery For Misinformation Detection

The PROMID shared task aims to explore methods for identifying or reconstructing the input prompt that likely led to the generation of a given piece of text, particularly in the context of misinformation generated by Large Language Models (LLMs).

Motivation

As LLMs become more prevalent, understanding how misinformation is generated is crucial. If we can trace back or infer the prompt that generated a suspicious text, it could provide insights into the intent, source, or specific instructions used to create misleading content. This can aid in developing more robust misinformation detection and mitigation strategies.

Task Overview

Given a piece of text suspected to be LLM-generated misinformation, participants are tasked to:

  • Subtask 1: Prompt Similarity Ranking: Given the target text and a set of candidate prompts, rank the candidate prompts by their likelihood of having generated the target text.
  • Subtask 2 (Optional): Prompt Generation: Generate a plausible prompt that could have produced the target text. Evaluation for this subtask will be more complex, possibly involving human evaluation or generation metrics.

Participants can choose to participate in one or both subtasks.
